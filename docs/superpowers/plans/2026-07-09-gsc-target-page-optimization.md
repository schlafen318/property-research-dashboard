# GSC Target Page Optimization Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Optimize the three Search Console target pages for query match, CTR, and internal routing.

**Architecture:** Keep all changes inside the existing static builder, then regenerate `artifacts/`. Add two small HTML helper functions and use existing destination/page data and tracking attributes.

**Tech Stack:** Python 3 standard library, static HTML generated by `src/build_unified_app.py`, generated output under `artifacts/`.

## Global Constraints

- Do not create new landing pages.
- Do not rename or remove existing URLs.
- Do not make legal, tax, visa, rental-income, or guaranteed-return claims.
- Do not broaden this batch to unrelated pages.
- Keep URL slugs, canonicals, schema structure, and sitemap URL count unchanged.
